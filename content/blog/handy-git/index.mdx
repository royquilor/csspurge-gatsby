Change origin of your GIT repository

```git
  git remote rm origin
  git remote add origin git@github.com:aplikacjainfo/proj1.git
  git config master.remote origin
  git config master.merge refs/heads/master
```

how to remote pull a branch?

dont do this.
git pull origin branch-name

this will just add commits to the master branch

to reset or set your branch to the remote branch
git fetch origin
git reset --hard origin/master

to see branches
Use git branch -a (both local and remote branches) or git branch -r (only remote branches)
